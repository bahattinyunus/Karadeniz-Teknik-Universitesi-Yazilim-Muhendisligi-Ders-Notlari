## 🏛️ VİZYON
Artık kod yazmayı biliyorsunuz; şimdi **sistem tasarlamayı** öğreneceksiniz.
Bu yıl, büyük resmi gördüğünüz yıldır. Yazılım Mimarisi ile sistemleri ölçeklendirmeyi, Yapay Zeka ile makinelere akıl vermeyi, Ağlar ile dünyayı birbirine bağlamayı öğrenirsiniz. Öğrencilik biter, Mühendislik başlar.

*   **Gizli Hedef:** "Coder" gömleğini çıkarıp, "Architect" pelerinini takmak.
